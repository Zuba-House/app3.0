# 🚀 How to Run Each Part of the Application

## ⚠️ **Important: Run from the correct directory!**

Each part of the application has its own `package.json` and must be run from its directory.

---

## 📱 **Mobile App**

```bash
cd mobile
npm start
```

Or use the auto-run script:
```bash
cd mobile
npm run android:auto
```

---

## 🖥️ **Backend Server**

```bash
cd server
npm start
```

For development:
```bash
cd server
npm run dev
```

---

## 🌐 **Client (Customer Frontend)**

```bash
cd client
npm start
```

---

## 👨‍💼 **Admin Dashboard**

```bash
cd admin
npm start
```

---

## 🏪 **Vendor Dashboard**

```bash
cd vendor
npm start
```

---

## 🎯 **Quick Reference**

| Application | Directory | Command |
|------------|-----------|---------|
| Mobile App | `mobile/` | `npm start` |
| Backend | `server/` | `npm start` |
| Client | `client/` | `npm start` |
| Admin | `admin/` | `npm start` |
| Vendor | `vendor/` | `npm start` |

---

## 💡 **Tip**

Always `cd` into the correct directory first, then run `npm start`!
