## ✅ **EASIEST: Use Your Phone (2 Minutes)**

### **Step 1: Install Expo Go**

- iOS: https://apps.apple.com/app/expo-go/id982107779

### **Step 2: Run This Command**
```powershell

```

- Open Expo Go app
- Tap "Scan QR Code"
- Point at terminal
- App loads! ✅

## 🔧 **For Standalone APK (15 Minutes)**

### **Fix SDK Build-Tools:**

1. **Open Android Studio**
2. **File → Settings → Android SDK**
3. **SDK Tools** tab
4. Check: **Android SDK Build-Tools 33.0.1**
5. Click **Apply** → Wait for install
6. Run: `npm run android`
